# Game Dev at GDC 2022

지난 2022년 3월 개최된 [Game Developers Conference](https://gdconf.com)에서 마이크로소프트가 공유한 [발표 및 영상](https://aka.ms/gamedev-at-gdc)에 대한 주요 한글 자료를 보실 수 있습니다.
